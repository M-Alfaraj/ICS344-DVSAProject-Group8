## Project Summary
This repository contains our ICS344 project deliverables for the DVSA (Damn Vulnerable Serverless Application) security assessment.

The project documents multiple vulnerabilities identified in the application, explains how each issue was reproduced, shows the impact, and presents the applied fixes and post-fix verification.
